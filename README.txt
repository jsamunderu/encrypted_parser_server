This is a server and client that evaluates simple math expressions consisting of only
digits, decimal point and */+-.

To build run mvn clean compile package

To run the test to verify the math expression correctness run mvn test

For ease of use run the jars with the dependencies embedded

Start the server then start the client

To evaluate an expression on the server through the client, you need to firts type CMD
followed by the math expression. Below are the commands to run.

CMD [simple math expression]
EXIT

The server and client uses end to end encryption
